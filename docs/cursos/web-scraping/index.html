Web Scraping

O objetivo deste curso é apresentar as principais ferramentas e estratégias para baixar e organizar dados da internet. Para isso, abordamos as noções básicas de como um site é construído, como funcionam as requisições web e como descobrir o caminho até um conteúdo específico de uma página. Também abordamos o uso de APIs e o que fazer quando o fluxo usual de raspagem não funciona e páginas dinâmicas usando Selenium.

Você sairá deste curso apta(o) a:

  • Organizar um projeto de web scraping;
  • Escrever scripts para baixar dados da internet;
  • Construir robôs para coletar dados da internet de forma automática;
  • Estruturar dados de arquivos baixados;
  • Baixar dados de APIs;
  • Usar ferramentas para baixar dados de sites dinâmicos;
