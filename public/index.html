Por que programar?

A programação é uma habilidade essencial para a pessoa cientista de dados. Listamos os principais motivos para você aprender a programar (em R).

O código é texto

A simples possibilidade de copiar e colar um código traz ganhos enormes em produtividade e reprodutibilidade.

O código é uma linguagem

O código é um diário de bordo. Ele documenta todas as etapas da análise de dados, algo essencial para reproduzirmos e compartilharmos nosso trabalho.

O código é aberto

Mais do que gratuito, ser aberto significa ser uma comunidade. Tenha acesso às últimas versões das melhores ferramentas e contate as pessoas criadoras para contribuir e tirar dúvidas.

O R é a linguagem criada para Ciência de Dados

O R nasceu dentro da Estatística e possui o ferramental necessário para executarmos todas as etapas de uma análise de dados.
